# 常见错误解决手册

## 🐛 错误1：backward() 重复调用

### 错误信息
```
RuntimeError: Trying to backward through the graph a second time
```

### 原因
在Jupyter中多次运行了包含`backward()`的cell

### 解决方法

**方法1：重新运行整个cell**
```python
# 把创建变量和backward()放在同一个cell
import torch

x = torch.tensor([2.0], requires_grad=True)
y = x ** 2 + 3 * x
y.backward()
print(x.grad)

# 这样无论运行多少次都不会错
```

**方法2：Restart Kernel**
- 点击上方 `Kernel` → `Restart`
- 重新运行

## 🐛 错误2：CUDA out of memory

### 错误信息
```
RuntimeError: CUDA out of memory
```

### 原因
GPU显存不够了（8GB用完了）

### 解决方法

**方法1：减小batch_size**
```python
# 原来
batch_size = 64

# 改成
batch_size = 16  # 或更小
```

**方法2：清空GPU缓存**
```python
import torch
torch.cuda.empty_cache()
```

**方法3：使用混合精度**
```python
from torch.cuda.amp import autocast

with autocast():
    output = model(input)
```

**方法4：重启Jupyter Kernel**
- Kernel → Restart

## 🐛 错误3：维度不匹配

### 错误信息
```
RuntimeError: mat1 and mat2 shapes cannot be multiplied
```

### 原因
矩阵乘法维度不对

### 解决方法

**检查维度：**
```python
print(x.shape)  # torch.Size([3, 4])
print(y.shape)  # torch.Size([5, 6])

# 如果要相乘，x的列数(4)必须等于y的行数
# 3x4 @ 4x6 = 3x6 ✅
# 3x4 @ 5x6 = 错误 ❌
```

**修正：**
```python
# 转置
z = torch.matmul(x, y.transpose(0, 1))

# 或改变形状
y = y.view(4, -1)
```

## 🐛 错误4：模型不在GPU上

### 错误信息
```
RuntimeError: Expected all tensors to be on the same device
```

### 原因
数据在GPU，模型在CPU（或相反）

### 解决方法

**确保都在GPU：**
```python
# 模型移到GPU
model = model.cuda()

# 数据也移到GPU
data = data.cuda()
labels = labels.cuda()

# 或统一用device
device = torch.device('cuda')
model = model.to(device)
data = data.to(device)
```

## 🐛 错误6：数据下载失败

### 错误信息
```
URLError: Connection timeout
```

# 多运行几次

- 下载到 `./data` 文件夹

## 🐛 错误7：Jupyter无法导入torch

### 错误信息

## 📋 调试技巧

### 1. 打印形状
```python
print(x.shape)  # 看张量大小
print(type(x))  # 看数据类型
```

### 2. 打印中间结果
```python
output = model(input)
print("模型输出：", output)
print("输出形状：", output.shape)
```

### 3. 检查数据范围
```python
print(x.min(), x.max())  # 看数据范围
print(x.mean(), x.std())  # 看分布
```

### 4. 使用断点调试
```python
import pdb
pdb.set_trace()  # 代码会在这里暂停
```
