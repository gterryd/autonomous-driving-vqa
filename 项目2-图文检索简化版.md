## 💻 核心功能代码

### 文本搜图

```python
# 简化版代码（完整的到时候给你）

from transformers import CLIPModel, CLIPProcessor
from PIL import Image

# 加载CLIP
model = CLIPModel.from_pretrained("openai/clip-vit-base-patch32")
processor = CLIPProcessor.from_pretrained("openai/clip-vit-base-patch32")

# 编码图片
image = Image.open("dog.jpg")
image_input = processor(images=image, return_tensors="pt")
image_embed = model.get_image_features(**image_input)

# 编码文本
text = "a dog on the beach"
text_input = processor(text=text, return_tensors="pt")
text_embed = model.get_text_features(**text_input)

# 计算相似度
similarity = (image_embed @ text_embed.T).item()
print(f"相似度: {similarity:.4f}")
```
