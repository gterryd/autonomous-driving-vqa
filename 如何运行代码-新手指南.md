# 如何运行代码 - 新手完全指南

## 方法1：使用Jupyter Notebook（推荐新手）

### 第1步：打开Anaconda Prompt

**Windows操作：**
1. 按键盘上的 `Win键`（Windows图标键）
2. 输入：`anaconda prompt`
3. 看到 "Anaconda Prompt" 程序
4. **右键点击** → 选择"以管理员身份运行"
5. 会弹出一个黑色窗口（命令行窗口）

**窗口显示：**
```
(base) C:\Users\29208>
```

### 第2步：激活学习环境

**在黑色窗口中输入（然后按回车）：**
```bash
conda activate multimodal
```

**窗口会变成：**
```
(multimodal) C:\Users\29208>
```

看到前面有`(multimodal)`就对了！

### 第3步：进入项目文件夹

**输入（然后按回车）：**
```bash
cd C:\Users\29208\Desktop\规划\项目学习\multimodal-ai-learning
```

**窗口会变成：**
```
(multimodal) C:\Users\29208\Desktop\规划\项目学习\multimodal-ai-learning>
```

### 第4步：启动Jupyter Notebook

**输入（然后按回车）：**
```bash
jupyter notebook
```

**会发生什么：**
1. 命令行窗口会显示一堆信息（不要关闭这个窗口！）
2. 浏览器会自动打开
3. 显示一个文件管理界面

**浏览器界面显示：**
```
Jupyter
  Notebook

Files   Running   Clusters

□ 01-foundations
□ 02-core-multimodal
□ 03-advanced
...
```

### 第5步：创建新的代码文件

**在浏览器的Jupyter界面中：**
1. 点击右上角的 `New` 按钮
2. 选择 `Python 3`
3. 会打开一个新的标签页

**你会看到：**
```
一个空白的格子（叫做"单元格"或"cell"）
In [ ]:
```

### 第6步：运行第一段代码

**在空白格子里，输入：**
```python
import torch
print("Hello PyTorch!")
print(f"PyTorch版本: {torch.__version__}")
print(f"CUDA可用: {torch.cuda.is_available()}")
```

**然后按：**
- 方法1：按键盘 `Shift + Enter`（推荐）
- 方法2：点击上方工具栏的 ▶️ 按钮

**会输出：**
```
Hello PyTorch!
PyTorch版本: 2.5.1
CUDA可用: True
```

**恭喜！你运行了第一段代码！**

### 第7步：运行学习文件中的代码

**打开学习文件：**
1. 用记事本/VSCode打开 `01-foundations/day01-05-pytorch-cnn.md`
2. 找到代码块（被```python```包围的部分）
3. 复制代码
4. 粘贴到Jupyter的格子里
5. 按 `Shift + Enter` 运行

**例如，复制这段：**
```python
# 创建一个3x4的随机矩阵
x = torch.randn(3, 4)
print("创建一个3x4的随机矩阵：")
print(x)
```

**粘贴到Jupyter，按Shift+Enter，会看到输出！**

### 第8步：保存你的代码

**在Jupyter中：**
1. 点击上方 `File` → `Save As...`
2. 输入文件名，比如：`day01_practice.ipynb`
3. 点击保存

**文件会保存在项目文件夹中**

### 第9步：关闭Jupyter

**关闭时：**
1. 关闭浏览器标签页
2. 回到黑色命令行窗口
3. 按两次 `Ctrl + C`
4. 输入 `y` 然后回车
5. 窗口可以关闭了

cd C:\Users\29208\Desktop\规划\项目学习\multimodal-ai-learning

Hello PyTorch!

## 方法3：使用VSCode（最专业）

### 安装VSCode

1. 访问：https://code.visualstudio.com/
2. 下载Windows版本
3. 安装（一路下一步）

### 配置VSCode

**第1步：打开项目文件夹**
1. 打开VSCode
2. 点击 `File` → `Open Folder`
3. 选择 `C:\Users\29208\Desktop\规划\项目学习\multimodal-ai-learning`
4. 点击"选择文件夹"

**第2步：安装Python插件**
1. 点击左侧的扩展图标（四个方块）
2. 搜索：`Python`
3. 点击安装（Microsoft官方的那个）
4. 搜索：`Jupyter`
5. 也安装上

**第3步：选择Python环境**
1. 按 `Ctrl + Shift + P`
2. 输入：`Python: Select Interpreter`
3. 选择 `multimodal` 环境

**第4步：创建并运行代码**
1. 点击 `File` → `New File`
2. 保存为 `test.py`
3. 输入代码
4. 右键点击编辑器 → 选择 `Run Python File in Terminal`

## ❓ 常见问题

**Q1: Shift + Enter没反应？**
A: 
- 确保光标在格子里
- 或点击上方▶️按钮

**Q2: 报错了怎么办？**
A:
- 看错误信息最后一行
- Google搜索错误
- 或复制错误问我

**Q3: 怎么停止运行？**
A:
- 点击上方 ⬛ 按钮
- 或菜单 Kernel → Interrupt

**Q4: 代码在哪里？**
A:
- 在Jupyter浏览器里写
- 自动保存在项目文件夹
- 文件名：Untitled.ipynb

**Q5: 怎么关闭Jupyter？**
A:
- 关闭浏览器标签
- 回到黑色窗口
- 按 Ctrl+C 两次
- 输入 y 回车

## 💡 温馨提示

### Jupyter快捷键
- `Shift + Enter`：运行当前格子，跳到下一个
- `Ctrl + Enter`：运行当前格子，不跳
- `Esc`：退出编辑模式
- `A`：在上方插入新格子
- `B`：在下方插入新格子
- `DD`（按两次D）：删除格子

### 保存工作
- Jupyter会自动保存
- 也可以手动：`Ctrl + S`
