### project1: 图文检索引擎
- 技术：CLIP + FAISS
- 时间：15天
- 难度：中等
- 适合：多模态/AI工程岗位

### project2: 智能相册助手
- 技术：CLIP + BLIP + FaceNet
- 时间：15天
- 难度：中等
- 适合：CV/多模态岗位

### project3: 自动驾驶场景理解
- 技术：VLM + nuScenes
- 时间：20天
- 难度：较高
- 适合：CV/自动驾驶岗位
